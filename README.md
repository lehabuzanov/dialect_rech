# Диалектная речь: Streamlit-приложение

Локальное и облачное Streamlit-приложение для:

- загрузки аудиофайла;
- прослушивания исходного аудио;
- получения орфографической расшифровки;
- построения отдельной широкой фонетической транскрипции;
- скачивания результатов.

## Структура

```text
dialect_rech/
├─ app.py
├─ README.md
├─ requirements.txt
├─ .gitignore
└─ src/
   ├─ __init__.py

```

## Стек

- интерфейс: `Streamlit`
- ASR: `faster-whisper`
- модели: direct CTranslate2-модели с Hugging Face
- визуализация аудио: `Plotly`
- чтение аудио без системного `ffmpeg`: `PyAV`

## Модели распознавания

В приложении есть реальные переключаемые модели:

- `Turbo: быстро и качественно`
- `Large v3: максимум качества`
- `Medium: баланс скорости и точности`
- `Small: самый лёгкий режим`

Переключение не декоративное: при распознавании используется именно выбранная модель.

## Что делает приложение

1. Пользователь загружает `wav`, `mp3`, `flac`, `ogg` или `m4a`.
2. Приложение показывает имя файла, размер, длительность и частоту дискретизации.
3. Пользователь может прослушать исходный файл прямо в интерфейсе.
4. Отображается волна аудио для визуального контроля.
5. После нажатия `Распознать` строится орфографическая запись.
6. На основе орфографии создаётся отдельная широкая фонетическая транскрипция.
7. Оба поля можно вручную редактировать.
8. Результаты можно скачать в `.txt` и `.json`.

## Установка

Рекомендуется Python `3.11+`.

```bash
git clone https://github.com/lehabuzanov/dialect_rech.git
cd dialect_rech
python -m venv .venv
```

Windows:

```bash
.venv\Scripts\activate
```

Linux/macOS:

```bash
source .venv/bin/activate
```

Установка зависимостей:

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

## Запуск

```bash
streamlit run app.py
```

## Ограничения текущей версии

- транскрипция строится rule-based, а не нейросетевой фонетической моделью;
- на очень тяжёлых или шумных файлах модель `Large v3` может быть медленной;
- для Streamlit Cloud обычно наиболее стабильна модель `Medium`.

## Возможные проблемы

### Если тяжёлая модель падает или зависает

Переключитесь на:

- `Turbo: быстро и качественно`
- `Medium: баланс скорости и точности`
- `Small: самый лёгкий режим`

### Если файл не читается

- проверьте формат;
- попробуйте пересохранить аудио;
- проверьте, что файл не повреждён.

### Если первая загрузка модели долгая

Это нормально: модель скачивается в кэш при первом запуске.
