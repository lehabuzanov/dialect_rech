# Диалектная речь: стабильное Streamlit-приложение

Приложение предназначено для расшифровки русской звучащей речи из загруженного аудиофайла.

Что умеет:

- загружать `wav`, `mp3`, `flac`, `ogg`, `m4a`;
- показывать информацию о файле и звуковую волну;
- воспроизводить исходный аудиофайл;
- распознавать речь через `faster-whisper`;
- выдавать орфографическую запись;
- строить отдельную транскрипцию;
- переключать тему интерфейса:
  - тёмно-синяя
  - светлая
- переключать формат транскрипции:
  - `Русская практическая`
  - `IPA / международная`
- скачивать результаты.

## Модели распознавания

В приложении оставлен один основной практичный режим:

- `Стабильная модель`

Это сделано специально для надёжной работы в Streamlit Community Cloud без падений на тяжёлых моделях.

## Транскрипция

Транскрипция строится через библиотечный движок `gruut`:

- `IPA / международная` строится напрямую по фонемам;
- `Русская практическая` получается как читаемое русскоязычное отображение той же фонемной записи.

## Установка

```bash
git clone https://github.com/lehabuzanov/dialect_rech.git
cd dialect_rech
python -m venv .venv
```

Windows:

```bash
.venv\Scripts\activate
```

Linux/macOS:

```bash
source .venv/bin/activate
```

Установка зависимостей:

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

## Запуск

```bash
streamlit run app.py
```

## Streamlit Community Cloud

Параметры деплоя:

## Краткая логика работы

1. Загрузите аудиофайл.
2. Выберите режим распознавания.
3. Нажмите `Распознать`.
4. При необходимости выберите нужный временной фрагмент.
5. Дождитесь прогресса до `100%`.
6. При необходимости переключите формат транскрипции.
7. Скачайте результат.

## Практические замечания

- используйте встроенную `Стабильную модель`;
- можно ограничить распознавание только нужным диапазоном времени;
- первая загрузка модели может занимать время, потому что веса скачиваются в кэш.
